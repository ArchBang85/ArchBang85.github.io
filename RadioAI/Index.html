    About the Radio Box Controller

    The box controller was built as part of the Alt Ctrl Game Jam 2014 by Petri Autio and Robin Baumgarten, along with a game where you fly a zeppelin on Venus. The entry won the competition and has been exhibited at Berlin's Game Science Center and at Alt.Ctrl.GDC The controller is based on a retro aviator radio box found at a London car boot sale. All of the radio controls have been connecetd to an Arduino which has a bluetooth connection to a Nexus 7 embedded in the top portion, forming a self-contained game device.
